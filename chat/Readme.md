# Chat server and client with Python Socket IO
This app demonstrate how to create simple chat server and client using Python Socket Io and PyQT5

# Screenshot

# Requirements
* Python 3+
* eventlet
* PyQt5
* PyYAML
* python-socketio

# Server (server/)
* Run "python chat-server.py"

# Client (client/)
* Run "python chat-client.py"
